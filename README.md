# Expense Tracker

A Java-based console application to record, categorize, and track personal expenses, helping users manage their finances effectively.

## Features

- Add, edit, and delete expense transactions
- Categorize expenses (e.g., food, transport, bills, entertainment)
- View daily, weekly, and monthly expense summaries
- Secure storage of transaction history
- Export and import data using CSV files
- Basic analytics for spending trends

## Technical Concepts Demonstrated

- Object-Oriented Programming (Encapsulation, Inheritance, Abstraction, Polymorphism)
- Exception Handling (try-catch-finally, custom exceptions)
- File I/O (reading/writing CSV files)
- Java Stream API for data processing
- Date/Time API for transaction timestamps
- Collections Framework (ArrayList, HashMap)
- Interfaces and Abstract Classes
- Lambdas and Functional Interfaces
- Recursion (optional utility methods)

## Java Architecture

- **JDK** (Java Development Kit): Development tools including compiler, debugger, and documentation
- **JRE** (Java Runtime Environment): Environment to run Java applications
- **JVM** (Java Virtual Machine): Executes Java bytecode and ensures platform independence

## Installation and Setup

### Windows Installation

1. Download JDK from Oracle's website
2. Run the installer and follow the prompts
3. Set JAVA_HOME environment variable to the JDK installation path
4. Add `%JAVA_HOME%\bin` to PATH environment variable

### IDE Setup (Eclipse/IntelliJ)

1. Download and install your preferred Java IDE
2. Create a new Java project
3. Configure the build path to use the installed JDK
4. Add project files and folders as per the structure

## How to Run

1. Clone the repository
2. Open the project in your IDE
3. Ensure Java 8 or later is configured
4. Run the `Main.java` class

## Sample Usage

The application provides a menu-driven interface to:

1. Add, edit, or delete expenses
2. Categorize expenses
3. View summaries by day, week, or month
4. Import/export transaction data
5. Analyze spending trends
